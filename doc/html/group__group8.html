◆ iteAudioActiveEngine()

int32_t iteAudioActiveEngine ( void  )

Active audio engine to start encode or decode except chosen MIDI engine.

Returns
0 if succeed, error codes of int32_t_ERROR otherwise.
Remarks
IF application choice decode function, audio engine will start to decode stream data inputted by iteAudioWriteStream(). IF application choice encode function, audio engine will start to record and encode data , and them application can use iteAudioReadStream() to get encoded data. IF application choice MIDI engine, iteAudioActiveMidiEngine() is desired to active midi engine.

◆ iteAudioReadStream()

unsigned long iteAudioReadStream ( uint8_t *  stream,
unsigned long  streamBufferSize,
unsigned long  streamthreshold 
)

Get encoded stream data from audio engine.

Parameters
streamSpecifies stream data buffer.
streamBufferSizeSpecifies stream buffer size.
streamthresholdSpecifies threshold value to avoid waitting.
Returns
encoded size.
Remarks
Audio engine record and encode data according to audio engine type chosen by iteAudioOpenEngine() API. Application can specify threshold value to avoid waitting, if the encoded data is smaller than threshold, the function will not get enocded data and return zero.
