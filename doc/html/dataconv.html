Data Converter

Description

This is a tool for converting data. It can:

  • Convert binary file to hex format text file.
  • Convert bitmap file to simple binary file. The format is:
4 bytes 4 bytes 4 bytes 4 bytes pitch * height
size width height format data

where size = (4 * 3) + (pitch * height) and format is

  • RGB565 = 0
  • ARGB1555 = 1
  • ARGB4444 = 2
  • ARGB8888 = 3
  • MONO = 4

Convert bitmap file to hex format text file.

Usage

dataconv [options]
-h, --help
show usage info
-q, --quiet
be quiet (no status messages)
-d, --debug
debug mode
-r, --raw
raw file
-o <file>, --output <file>
output file
-x <file>, --bin2hex <file>
convert binary to hex
-b <file>, --bmp2bin <file>
convert bmp to binary
-1, --format=RGB565
convert to RGB565 image
-2, --format=ARGB8888
convert to ARGB8888 image
-i, --dither
enable dithering
-m <file>, --bmp2hex <file>
convert bmp to hex

Examples

  • Convert binary file test.bin to hex text file test.hex
    dataconv -x test.bin
  • Convert binary file test.bin to hex text file abc.txt
    dataconv -x test.bin -o abc.txt
  • Convert bitmap file test.bmp to simple binary file test.bin
    dataconv -b test.bmp
  • Convert bitmap file test.bmp to simple binary file abc.bin
    dataconv -b test.bmp -o abc.bin
  • Convert bitmap file test.bmp to hex text file abc.txt
    dataconv -m test.bmp -o abc.txt
