SPI

This is the driver for spi.

The driver sources are put at /sdk/driver/spi/mmp_spi.c

  • Functions
    1. ioctl()

      1. mmpSpiInitialize()

        File system must call this API first when initializing a spi bus.

      2. mmpSpiTerminate()

        This routine is used to release any resources associated with a drive when it is terminated.

    2. Read() and Write()

      1. mmpSpiDmaWrite()

        This routine is used to write data to targe device by dma.

      2. mmpSpiDmaWriteMulti()

        write multi data to targe device by dma.

      3. mmpSpiDmaRead()

        This routine is used to read data from targe device by dma.

      4. mmpSpiDmaTriggerRead()

        Trigger the dma read.

      5. mmpSpiPioWrite()

        This routine is used to write data to targe device by pio.

      6. mmpSpiPioWriteNoLock()

        May control the lock when writing data to targe device by pio.

      7. mmpSpiPioRead()

        This routine is used to read data from targe device by pio.

    3. Set Mode()

      1. mmpSpiSetMode()

        Set the mode for spi.

      2. mmpSpiSetMaster()

        Set the mode for spi master.

      3. mmpSpiSetSlave()

        Set the mode for spi slave.
