Booting Sequence

OpenRTOS

\openrtos\boot\startup.S

  • Initialize interrupt vectors
  • Initialize MMU
  • Resume from suspend mode if possible
  • Clear bss section
  • Call BootInit()
  • Jump to main()

\openrtos\boot\init.c

  • Early initialize hardware drivers without OS

\sdk\share\sdl\main\openrtos\SDL_openrtos_main.c

  • Start main task
  • Initialize device drivers
  • Invoke the C++ static constructors
  • Start other tasks
  • Call to project provided SDL_main()

\sdk\driver\itp\itp_init_openrtos.c

  • Initialize device drivers
