Battery

This is the driver for battery detection.

The driver sources are put at /sdk/driver/itp/battery/*.c. Its parent driver is itp_power.c

  • Functions

    1. read()

      Reads power status. Will call to itpBatteryGetState() and itpBatteryGetPercent().

    2. ioctl()

      1. ITP_IOCTL_INIT

        Inizializes this module. Will call to itpBatteryInit().

  • GPIOs

    1. CFG_GPIO_BATTERY

      Defines the pin numbers of battery GPIO detection function.

    2. CFG_GPIO_BATTERY_CHARGE_DETECT

      Defines the pin number of battery GPIO charge detection function.

  • Configurations

    1. CFG_BATTERY_ENABLE

      Determines build/link this module or not.

    2. CFG_BATTERY_MODULE

      Determines which sdk/driver/itp/battery/itp_battery_<module>.c will be built/linked.

    3. CFG_BATTERY_CHARGE_DETECT

      Determines enable charge detection or not.
