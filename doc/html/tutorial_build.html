How to Build

  1. Copy ITEGCC toolchain to C:/ITEGCC

  2. Execute build/openrtos/<project>.cmd which you want

    (For example : test_posixapi.cmd )

  3. Tune the setting configuration you want

  4. Press the Save to save the project config

  5. Press the Build to build the project

  6. Wait the minutes until the build completed.

  7. During the building time, press the Stop to break the build

  8. Click the Run to start work ( The hardware need to be set up )

  9. To exit the program, press file->Quit
