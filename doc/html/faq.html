Frequently Asked Questions

How to re-configure project without rebuild project?

  • Delete build\<OS>\<project>\.config and config.cmake
  • Execute build\<OS>\<project>.cmd again

How to rebuild project without re-conifure?

  • Delete build\<OS>\<project>\*.* except .config and config.cmake
  • Execute build\<OS>\<project>.cmd again

How to re-conifure project from previous configured settings?

  • Delete build\<OS>\<project>\config.cmake
  • Execute build\<OS>\<project>.cmd again

How to overwrite default settings in my project?

  • Edit project\<project>\Kconfig
  • For example, to disable FAT, just add this item before source command as follow:
    ...
    config FS_FAT
    def_bool n
    source "$CMAKE_SOURCE_DIR/sdk/Kconfig"
    ...

Sometimes the project will build fail as follow messages:

Linking C executable <project>/cygdrive/c/ITEGCC/lib/gcc/arm-elf/4.6.2/../../../../arm-elf/bin/ld: cannot find -lopenrtos
collect2: ld returned 1 exit status
make[2]: *** [project/<project>/<project>] Error 1

By default, the SDK will use multi-thread function of make to build projects and sometimes it will build failed. You can:

  • Run the <project>.cmd again or
  • Edit <project>.cmd, add
    set MAKEJOBS=1
    to disable multi-thread function of make.

How to use RAM Disk?

  • Enables RAM Disk and FAT functions on Configuration menu.
  • Edit project\<project>\CMakeLists.txt, add copy commands like this:
    # font
    file(COPY
    ${PROJECT_SOURCE_DIR}/data/font
    DESTINATION ${CMAKE_BINARY_DIR}/data/private
    )
    configure_file(${PROJECT_SOURCE_DIR}/sdk/target/font/${CFG_FONT_CONF}.conf.in ${CMAKE_BINARY_DIR}/data/private/fonts.conf)
    # flash ui
    configure_file(${PROJECT_SOURCE_DIR}/data/flash_ui/main_${CFG_LCD_WIDTH}x${CFG_LCD_HEIGHT}.swf ${CMAKE_BINARY_DIR}/data/private/flash_ui/main.swf COPYONLY)
  • Build this project.
