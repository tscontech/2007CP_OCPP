Headset

This is the driver for headset detection.

The driver sources are put at /sdk/driver/itp/headset/*.c. Its parent driver is itp_headset.c

  • Functions

    1. read()

      Reads headset value. Will call to itpHeadsetGetValue().

    2. ioctl()
      1. ITP_IOCTL_INIT Inizializes this module. Will call to itpHeadsetInit().
  • GPIOs

    1. CFG_GPIO_HEADSET

      Defines the pin number of headset GPIO function.

  • Configurations

    1. CFG_HEADSET_ENABLE

      Determines build/link this module or not.

    2. CFG_HEADSET_MODULE

      Determines which sdk/driver/itp/headset/itp_headset_<module>.c will be built/linked.
