Uart

This is the driver for uart.

The driver sources are put at /sdk/driver/itp/itp_uart.c Its parent driver is itp_uart.c

  • Functions

    1. read()

      Uart rx.

    2. write()

      Uart tx.

    3. ioctl()

      1. ITP_IOCTL_INIT

        Inizializes this module. Will call to UartInit().

      2. FIONREAD

        Is a character waiting. Will call to ithUartIsRxReady().

  • GPIOs

    1. CFG_GPIO_UART1_TX

      Defines the pin number of uart1 GPIO tx.

    2. CFG_GPIO_UART1_RX

      Defines the pin number of uart1 GPIO rx.

  • Configurations

    1. CFG_UART1_ENABLE

      Determines build/link this module or not.

    2. CFG_UART1_BAUDRATE

      Determines uart baudrate.

    3. CFG_UART_INTR

      Determines Interrupt enable/disable.
