SoC官网注册

  1. 请至 http://soc.ite.com.tw 注册，注册后请提供以下数据，并通知FAE人员做账号开通。
    • 注册E-mail
    • 账号与公司名称
    • 应用类型
  2. 账号开通后，请在SMART CONTROL类别中下载SDK、HDK与教学视频。

下载 ITE IDE

请到Home->Smart Control->Software Design Kit中，下载 Toolchain 与 Tools 。

Toolchain

ITE 提供一个名为ITEGCC的GCC Based toolchain，您可在Home->Smart Control->Software Design Kit中找到。请根据您的host开发环境，下载Linux或Windows的版本。
